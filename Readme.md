# 题目:羽毛球馆

**运行环境**：
- **JDK 1.8**
- **Junit  4.12**
- **IntelliJ IDEA**
		
## 代码结构
- dao **数据接口**
  - impl **数据接口实现**
- entity **实体**
- Controller.java **控制类**
- Main.java **运行入口**
- service **业务层接口**
  - implService **业务层实现**
- util **工具**
- lib **junit依赖的相关jar包**
